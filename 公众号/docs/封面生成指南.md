# 公众号封面生成指南

## 封面设计规范

**尺寸**：900 × 383（头条封面比例 2.35:1）

**核心元素**：
- **标题**：文章标题，大字体，居中，醒目
- **日期**：YYYY.MM.DD 格式，明显可见，标题下方或角落
- **副标题**：公众号名"AI 随想录"，底部小字

**风格**：
- 以文字为主
- 深色背景 + 渐变装饰
- 标题用亮色（白色或浅色）
- 日期用较亮颜色（不能太小）

---

## 生成流程

### 1. 创建 SVG 源文件

```bash
# 位置：文章目录下
~/Documents/public-notes/公众号/YYYY-MM-DD-标题/cover.svg
```

### 2. SVG 模板

```svg
<svg width="900" height="383" xmlns="http://www.w3.org/2000/svg">
  <!-- 渐变背景 -->
  <defs>
    <linearGradient id="bg" x1="0%" y1="0%" x2="100%" y2="100%">
      <stop offset="0%" style="stop-color:#1a1a2e"/>
      <stop offset="100%" style="stop-color:#16213e"/>
    </linearGradient>
  </defs>
  
  <rect width="900" height="383" fill="url(#bg)"/>
  
  <!-- 标题（大字，居中） -->
  <text x="450" y="180" font-family="PingFang SC, sans-serif" font-size="64" 
        font-weight="bold" fill="white" text-anchor="middle">文章标题</text>
  
  <!-- 日期（明显可见） -->
  <text x="450" y="260" font-family="PingFang SC, sans-serif" font-size="32" 
        fill="#a8d8ea" text-anchor="middle">2026.03.30</text>
  
  <!-- 公众号名（底部） -->
  <text x="450" y="340" font-family="PingFang SC, sans-serif" font-size="16" 
        fill="#7f8c8d" text-anchor="middle">AI 随想录</text>
</svg>
```

### 3. 转换为 PNG

```bash
rsvg-convert -o cover.png cover.svg
```

### 4. 上传到微信

```bash
# 上传为永久素材
curl -X POST "https://api.weixin.qq.com/cgi-bin/material/add_material?access_token=$TOKEN&type=image" \
  -F "media=@cover.png"

# 返回 media_id，用于创建草稿
```

## 检查清单

生成前确认：
- [ ] 标题是否大且醒目？
- [ ] 日期是否明显可见（字号≥24）？
- [ ] 整体是否以文字为主？
- [ ] 背景和文字对比度足够？
